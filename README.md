Clothing Business CRM

A Customer Relationship Management (CRM) system designed for clothing retail businesses.

Project Overview

This project aims to help clothing stores manage customers, sales, invoices, inventory, and business operations more efficiently.

Features

- Customer Management
- Product Management
- Sales & Billing
- Invoice Generation
- Inventory Tracking
- Sales History

Tech Stack

- Frontend: Next.js
- Backend: Supabase
- Database: PostgreSQL
- Styling: Tailwind CSS

Status

🚧 Currently under development as an MVP.

Team

Developed as part of a university software engineering project.
